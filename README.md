# TASK-3-MICRO-INTERACTION-DESIGN-
Wishlist Heart Button Micro-interaction Design project completed during CODTECH IT Solutions UI/UX Internship.

## Company
CODTECH IT SOLUTIONS

## Domain
UI/UX Design

## Project Name
Wishlist Heart Button Micro-interaction

## Project Overview
This project demonstrates a micro-interaction design for a Fashion E-commerce Application wishlist feature.

The interaction provides visual feedback when users add a product to their wishlist.

## Objective
To improve user engagement and create a smooth interactive shopping experience.

## Features
- Default Heart State
- Click Interaction
- Smooth Animation
- Active Wishlist State
- User Feedback

## Interaction Flow
User Click → Animation → Active State → Confirmation Feedback

## Tools Used
- Figma
- UX Research
- Canva
